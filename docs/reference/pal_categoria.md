# Paleta categorica para graficos de colecciones y recursos

Paleta de colores fija utilizada en graficos categoricos (disciplinas y
sectores) del Registro de Colecciones de GBIF.ES.

## Usage

``` r
pal_categoria
```

## Format

Un vector de caracteres nombrado con codigos hexadecimales.

## Details

Los colores estan definidos de forma explicita para garantizar
consistencia visual entre graficas, independientemente de los datos.
