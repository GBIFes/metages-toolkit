# Inserta la tabla de colecciones en el informe Word

Genera los datos actualizados, aplica el formateo de valores, construye
la flextable con hipervinculos e inserta la tabla en el informe Word
generado previamente.

## Usage

``` r
insertar_tabla_colecciones(keyword)
```

## Arguments

- keyword:

  Texto exacto del encabezado donde insertar la tabla.

## Value

Ruta al archivo .docx modificado.
